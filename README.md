# AI-Powered Smartphone Sales Prediction System

An intelligent machine learning-based platform that analyzes smartphone market data and predicts future sales trends.  
The system combines predictive analytics, smartphone comparison tools, and a buy/sell marketplace to help businesses and consumers make data-driven decisions.

---

## Project Overview

The smartphone market evolves rapidly due to technological advancements, new product launches, and changing consumer preferences.  
Accurately forecasting demand is essential for manufacturers, retailers, and e-commerce platforms.

Traditional forecasting approaches often fail because they cannot capture complex relationships between multiple factors such as price, specifications, brand popularity, and consumer sentiment.

This project solves the problem by developing a machine learning-based system capable of predicting smartphone sales using historical data and market indicators.

In addition to forecasting, the system also provides:

- Smartphone comparison functionality  
- A marketplace for buying and selling used smartphones  
- Data visualization dashboards for sales insights  

---

## Objectives

- Predict smartphone demand using machine learning models  
- Identify factors that influence smartphone sales  
- Improve inventory management and production planning  
- Provide a smartphone comparison system  
- Enable a platform for buying and selling second-hand smartphones  
- Support data-driven decision making through visual analytics

## Key Features

### Smartphone Sales Prediction
Uses machine learning models trained on historical data to forecast future smartphone demand.

### Smartphone Comparison Tool
Allows users to compare smartphones based on specifications such as price, battery, camera, performance, and ratings.

### Buy/Sell Marketplace
Users can list second-hand smartphones for sale and purchase devices from other users.

### Data Visualization Dashboard
Provides graphical insights into market trends, sales predictions, and feature influence.

## Machine Learning Models Used

The system evaluates multiple machine learning models including:

- Linear Regression
- Random Forest
- Gradient Boosting
- LSTM Neural Networks

Each model is trained and evaluated using cross-validation and hyperparameter tuning.  
The best performing model is used for final predictions.
